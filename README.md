# 🎬 404 MoviesInfo

A modern Movie Discovery Web Application built with React, Firebase Authentication, and TMDB API.

Users can browse trending movies, search movies, view detailed movie information, watch trailers, explore actor profiles, and see movies acted by their favorite actors.

---

## 🚀 Features

### Authentication

* Email & Password Login
* Google Authentication
* Protected Routes
* Logout Functionality

### Movies

* Trending Movies
* Popular Movies
* Top Rated Movies
* Upcoming Movies
* Movie Search
* Movie Details Page
* Movie Genres
* Movie Cast Information
* Movie Trailers (YouTube Popup)

### Actors

* Actor Details Page
* Actor Biography
* Date of Birth
* Place of Birth
* Movies Acted By Actor

### UI Features

* Responsive Design
* Mobile Friendly
* Modern Movie Cards
* Carousel Banner
* Pagination
* Custom Theme Colors

## ⚙️ Environment Variables

Create a `.env` file in the root folder:

```env
VITE_TMDB_API_KEY=your_tmdb_api_key

VITE_FIREBASE_API_KEY=your_firebase_api_key
VITE_FIREBASE_AUTH_DOMAIN=your_auth_domain
VITE_FIREBASE_PROJECT_ID=your_project_id
VITE_FIREBASE_STORAGE_BUCKET=your_storage_bucket
VITE_FIREBASE_MESSAGING_SENDER_ID=your_sender_id
VITE_FIREBASE_APP_ID=your_app_id
```

## 📦 Installation

Clone the repository:

```bash
git clone https://github.com/kavin997007/-tmdb-movie-app.git
```

Navigate to project:

```bash
cd -tmdb-movie-app
```

Install dependencies:

```bash
npm install
```

Run project:

```bash
npm run dev
```
